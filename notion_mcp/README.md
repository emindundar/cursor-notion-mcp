# Notion MCP (Cursor Entegrasyonu)\n\nPython 3.12 ile Notion'a not, görev, kod parçacığı ekleyen, arama yapan ve Git özetleri sunan MCP sunucusu.\n\n## Kurulum\n1. Depo kökünde olun: `cd cursor-notion-mcp/notion_mcp`\n2. Sanal ortam (önerilir):\n   ```bash\n   python3 -m venv .venv\n   source .venv/bin/activate\n   ```\n3. Bağımlılıklar: `pip install -r requirements.txt`\n4. Ortam değişkenleri: `.env.example` dosyasını kopyalayın ve doldurun.\n   ```bash\n   cp .env.example .env\n   # NOTION_API_KEY ve NOTION_PAGE_ID alanlarını doldurun\n   ```\n\n## Çalıştırma\n```bash\npython server.py\n```\nSunucu başladığında MCP araçları yüklenmiş olur.\n\n## Cursor MCP Entegrasyonu\n`mcp.json` içine ekleyin (tam yolu güncelleyin):\n```json\n{\n  \"providers\": [\n    {\n      \"name\": \"NotionAssistant\",\n      \"command\": \"python /full/path/to/cursor-notion-mcp/notion_mcp/server.py\"\n    }\n  ]\n}\n```\n\n## Sağlanan Araçlar\n- `add_note(content, title=None)`\n- `add_todo(task)`\n- `save_code_snippet(code, language=\"python\", description=\"\")`\n- `search_in_notion(query)`\n- `get_git_summary(since=\"6am\", project_path=None)`\n\nAPI anahtarı ve sayfa izni sağlandığında kullanıcıdan ek kurulum beklenmez.\n*** End Patch
